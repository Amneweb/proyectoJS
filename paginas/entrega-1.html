Bienvenidos a Cine VINTAGE

Este proyecto es un sitio web para un cine, que incluye una aplicación para la venta de entradas y snacks. Para la venta de entradas el usuario elegirá, en el siguiente orden:

  1. Película
  2. Tipo de sala (premium, clásica, infantil)
  3. Función (fecha y hora)
  4. Cantidad de entradas
  5. Asientos

La elección de los asientos se hará en función de la ubicación de cada asiento en la sala (se mostrará una especie de plano con la disposición de las butacas y el usuario hará click en el asiento elegido). Según la ubicación del asiento, éste tendrá un valor específico

Seleccionados los asientos, el usuario podrá elegir si quiere agregar snacks o si sólo compra las entradas.

Primer entrega

Para esta primer entrega del proyecto, acá mostramos las opciones para que el usuario ingrese en el prompt la opción elegida. Con ese dato calculamos el total y lo mostramos en el alert.
